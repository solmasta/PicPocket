# PicPocket
PicPals is a fun, easy-to-use photo storage app that keeps your memories safe. Snap, tag, and share your favorite moments with automatic backups to Google Photos and Drive. Create collages, stories, and slideshows, or use AI to auto-tag your photos. Perfect for capturing and reliving all your special moments.

## Ranch gallery demo
- Open `/home/runner/work/PicPocket/PicPocket/index.html` in a browser.
- Includes:
  - Stable door 2–3 column gallery grid
  - Horse hoof print loading animations
  - Fence post separators between photos
  - Barn window frame styling
  - Hover overlays with horse names and tags

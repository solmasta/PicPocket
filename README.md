# PicPocket

PicPals is a fun, easy-to-use photo storage app that keeps your memories safe.
Snap, tag, and share your favorite moments with automatic backups to Google
Photos and Drive. Create collages, stories, and slideshows, or use AI to
auto-tag your photos.

## Getting started

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm start
```
